# Heroes and Villains

Heroes and Villains is a single player adventure game where players can create a party of super heroes to rid each city of it's dangerous villain.

## Getting Started

These instructions will describe how to build the source code and run the game.

### Prerequisites

If the following resources are not already installed, please consider installing them.

```
Version 8 Update 171
https://java.com/en/download/
```
```
Java SE Development Kit 8 Downloads
http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
```

## Running the game

### Using a .jar file

To run the executable JAR file for the Heroes and Villains adventure game, simply open a terminal and navigate to the directory:

```
cd ...\HeroesAndVillains
```

Once at this directory, perform the following command:

```
java -jar aco108-frd15-HeroesAndVillains.jar
```

The main menu screen GUI for Heroes and Villains should now be visible, and the game is now playable.

### Using eclipse

1. Open a new eclipse work session and import the HeroesAndVillians folder to create a new java project.
2. Navigate to the GuiMain.java file, located in "src/gui/".
3. Click "Run > Run" in the top window bar. Alternatively, on Windows, press Ctrl + F11 to run the game.

## JUnit Testing

All of the JUnit tests can be found in the folder at "src/junit".
